# Miaufajores

Sitio web desarrollado con HTML5 y CSS3 en el marco de la Pre-Entrega para el curso de Front End JS de Talento Tech.


## 📌 Características

- Estructura semántica HTML5
- Diseño responsive básico
- Catálogo de productos
- Formulario de contacto funcional con Formspree
- Navegación interna entre secciones
- Uso de Flexbox y Grid
- Implementación de Media Queries
- Integración de Google Fonts
- Contenido multimedia e imágenes de la marca


## 📁 Estructura

Pre-Entrega/
│
├── index.html
├── README.md
├── css/
│ └── styles.css
└── img/
  └── (imágenes de la marca)


## 🚀 Uso

1. Proyecto con fines de aprendizaje
2. Abrir `index.html` en el navegador


## 🛠️ Tecnologías

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
- Formspree
